# HR Attrition Analysis Dashboard 📊

This project presents an HR Attrition Analysis Dashboard built using Power BI.  
The dashboard helps analyze employee turnover trends and provides insights into workforce attrition patterns.

---

## 📌 Project Objective

The objective of this project is to understand employee attrition trends and identify key factors that contribute to employees leaving the organization.

---

## 🛠 Tools Used

- Power BI
- Data Cleaning
- Data Modeling
- DAX Measures
- Data Visualization

---

## 📊 Dashboard Features

The dashboard provides insights into:

- Total Employees
- Total Attrition Count
- Attrition Rate
- Department-wise Attrition
- Gender Distribution
- Age Group Analysis
- Job Role Analysis
- Interactive Filters (Slicers)
